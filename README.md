# ESP32 SSD1331 library BV2
This is Arduino core for the ESP32 library ( SPI ).  
Beta ver 2.0.0
HSPI only.  
  
(example)  
SCLK-----GPIO #14  
MOSI-----GPIO #13  
MISO-----NONE  
CS-------GPIO #15  
D/C------GPIO #21  
RST------GPIO #4  
  
Use SPI.h library
